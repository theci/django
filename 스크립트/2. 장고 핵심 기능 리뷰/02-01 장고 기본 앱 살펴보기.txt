자 이번 시간에는 장고 기본 앱이 어떤 것들이 있는지 간략히 살펴 보겠습니다
Python 패키지는 모듈하드 위에 연관된 Python 코드를 하나의 폴더에 모아놓은
것입니다. Django 앱은 Django 프로젝트에 사용하기 위한 Python 패키지로서
연관된 모델, 뷰, 템플릿, 테스트, 어드민 등을 하나의 폴더에 모아놓은 것 뿐입니다.

그렇기에 라이브러리를 설치하고 간단한 설정만으로 온전히 동작하는 기능을 경험할 수
있습니다. 이러한 모듈화가 장고에만 있는 개념이 아닙니다.
