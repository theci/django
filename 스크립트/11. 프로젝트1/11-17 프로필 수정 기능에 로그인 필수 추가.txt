로그아웃 상황에서 프로필 수정 페이지에 접근하시면 이렇게 AttributeError 예배가 발생하고 있습니다.
Anonymous user 객체에는 프로필 이름의 속성이 없다는 오류인데요.
Vue 소스 코드의 90번째 줄을 보시면 로그인 유저와 1대1 관계에 있는 프로필을 조회하는 코드가 있습니다.
그런데 로그아웃 상황에서는 Request 객체의 유저 속성이 UserModel 인스턴스가 아니라 일반 Python 클래스인 AnonymousUser 클래스의 인스턴스입니다.
로그인 상황에서만 유저 객체의 프로필 속성을 참조하실 수 있습니다.
먼저 getObject 메서드에서는 로그아웃 상황에서는 방어적으로 논을 즉시 반환도록 하겠고요. 
페이지 새로고침하시면 오류가 사라졌죠.
그런데 로그아웃 상황에서는 프로필 수정이 불가능해야 합니다.
그러니 프로필 수정에서는 로그인 상황임을 보장받기 위해서 LoginRequiredMixIn 클래스를 상속받도록 합니다.
페이지 새로고침하시면 로그인 페이지도 이동하고요.
로그인하시면 프로필 수정을 이어서 진행하실 수 있게 됩니다.
